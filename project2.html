About the Website

This is a professional, clean, and easy-to-navigate resume website designed to showcase my qualifications, projects, and achievements. Hosted on GitHub Pages, it serves as a central hub for potential employers and collaborators.

Key Features

  • Resume Accessibility: - Offers downloadable versions of my resume in PDF, DOCX, and a Hebrew version.
  • Work History: - Highlights my previous work history and experience
  • Responsive Design: - Ensures an optimal experience across desktops, tablets, and mobile devices.
  • User-Friendly Navigation: - Simple layout with quick access to essential information.
  • Dynamic Content: - Features live demos and engaging visuals to present my work effectively.

Technologies Used

  • Frontend: HTML5, CSS3
  • Hosting: GitHub Pages
